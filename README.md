# 📦 Qx Tools

Qx Tools est un petit outil en Python qui permet de générer automatiquement des structures de projets prêtes à l’emploi.

Il peut créer des bases pour :

* 🌐 projets web
* 🧠 projets IA
* 📁 projets simples / base
* ⚙️ et plus de templates à venir

---

# 🚀 Fonctionnalités

* Génération automatique de dossiers et fichiers
* Structure web complète (frontend + backend + API + DB)
* Structure de base Python + docs + SQL
* Structure IA prête à développer
* Interface simple en terminal (menu interactif)
* Installation automatique en mode CLI (optionnel)

# 🧩 Option CLI (Qx Tool install)

Le projet peut s’installer dans le système et être lancé partout :

* crée un dossier `qx_tool` dans l’utilisateur
* ajoute un fichier `.bat`
* ajoute le chemin au PATH Windows

---

# 🛠️ Technologies utilisées

* Python 🐍
* os (gestion fichiers)
* time (animations / délais)
* questionary (interface terminal)
* winreg (ajout PATH Windows)

---

# ⚠️ Important

* Fonctionne uniquement sur **Windows** (à cause de `winreg`)
* Nécessite Python installé
* Certaines fonctions modifient le PATH système

---

# 💡 Idée du projet

Qx Tools a été créé pour éviter de perdre du temps à recréer toujours les mêmes structures de projets.

Au lieu de faire :

* dossiers manuels
* fichiers de base
* setup répétitif

👉 tu fais tout en 1 commande / 1 menu.
